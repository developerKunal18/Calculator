# Calculator

## Overview
The **Calculator** is a simple Python command-line application that performs basic arithmetic operations.

It allows users to add, subtract, multiply, and divide numbers using a menu-driven interface.

---

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Input validation
- Menu-based interaction

---

## Technologies Used
- Python
- Functions
- Loops
- Conditional statements
- User input handling

---

## How It Works
1. User selects an operation from the menu  
2. User enters two numbers  
3. Calculator performs the selected operation  
4. Result is displayed  
5. User can continue or exit
